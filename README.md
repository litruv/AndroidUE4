# Ard4UE

An **Unreal Engine 4 Plugin** Implementation of the serial library by wjwwood https://github.com/wjwwood/serial

### Features include:

- String Parsing
- Float Parsing
- Write to Serial
- Events in Unreal and Arduino
- ID / Auto Connect depending on name
